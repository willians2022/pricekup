# pricekup
PriceKup transforma números em crescimento para o seu negócio de doces artesanais. Planeje receitas, calcule custos de ingredientes, defina preços de venda com precisão, acompanhe suas vendas e organize suas finanças em um só lugar. Tenha mais controle, aumente sua lucratividade e alcance suas metas financeiras com praticidade e confiança. 📈🧁💰🚀
